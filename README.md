# Xposed Framework

## Compatibility
* Android Lollipop - Oreo
* init.d (other root & rootless)
* MagiskSU & SuperSU
* Magisk & System install
* Nexus/Pixel support (A/B OTA)
* SELinux enforcing
* Works with nearly every device, kernel, and rom
